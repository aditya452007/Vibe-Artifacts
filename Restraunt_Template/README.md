# Restaurant Template (Oh Indore)

[← Back to Vibe Artifacts](../README.md)

A premium, modern restaurant website template built with React, Vite, and Tailwind CSS. This template features smooth animations, responsive design, and a polished user interface suitable for high-end dining experiences.

## Features

- **Modern Tech Stack**: Built with [React](https://react.dev/) and [Vite](https://vitejs.dev/) for blazing fast performance.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for utility-first styling with custom design tokens.
- **Animations**: Rich, interactive animations using [GSAP](https://gsap.com/) and [Framer Motion](https://www.framer.com/motion/).
- **Smooth Scrolling**: Integrated [Lenis](https://github.com/darkroomengineering/lenis) for buttery smooth scroll experiences.
- **Routing**: [React Router](https://reactrouter.com/) for seamless page navigation.
- **Icons**: [Lucide React](https://lucide.dev/) and [React Icons](https://react-icons.github.io/react-icons/).

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd Restraunt_Template
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Build

To build the project for production:

```bash
npm run build
```

## Project Structure

- `src/`: Source code including components, pages, and styles.
- `public/`: Static assets.
- `tailwind.config.js`: Tailwind CSS configuration.
- `vite.config.js`: Vite configuration.
