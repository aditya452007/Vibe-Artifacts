# Vibe Prompt Builder Extension

## Overview
Vibe Prompt Builder is a browser extension designed to help users construct and refine prompts for AI models. This tool allows for the creation, management, and export of prompts, streamlining the prompt engineering workflow.

## Features
- **Prompt Construction**: Easily build complex prompts using a modular interface.
- **Template Management**: Save and reuse prompt templates.
- **Export Options**: Export prompts to various formats.
- **Aesthetic UI**: A modern, clean interface designed for ease of use.

## Usage
1.  **Installation**:
    - Clone this repository.
    - Navigate to `Vibe-Prompt-Builder`.
    - Run `npm install` to install dependencies.
    - Run `npm run build` to build the extension.
    - Load the `dist` folder as an unpacked extension in Chrome/Edge.

2.  **Development**:
    - Run `npm start` or `npm run dev` (if available) for hot-reloading during development.
