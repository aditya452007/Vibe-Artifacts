## 🌟 Overview

**Time Manager** is a premium, native Windows application designed to help you reclaim your focus hours without sacrificing privacy. Unlike cloud-based trackers, Time Manager runs entirely locally, ensuring your productivity data remains yours alone.

## 🚀 Features

*   **⚡ Zero-Touch Tracking**: Our intelligent daemon monitors active window focus events, distinguishing between work and distraction automatically. No manual timers needed.
*   **🔒 Local Encryption**: Your data is stored strictly on your local disk using **SQLite with AES-256 encryption**.
*   **🏎️ Native Performance**: Written in **Rust**. Consumes < 0.1% CPU and ~25MB RAM. You won't even feel it running.
*   **🔌 100% Offline**: No accounts, no cloud sync, no data selling.

## 🛠️ Installation

1.  **Download**: Get the latest version from the [Releases Page](https://github.com/aditya452007/Vibe-Artifacts/releases/download/V1/TimeManager.zip).
2.  **Unzip**: Extract the contents of the zip file to your preferred location.
3.  **Run**: Execute `TimeManager.exe`.

*   **Database**: SQLite (Encrypted)
*   **UI**: Native Windows UI
